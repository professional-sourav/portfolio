About Me

I am Sourav Chakraborty, a Full-Stack Developer from India with over 8 years of experience in software and web development. My core strengths lie in Core PHP, Laravel, WordPress, JavaScript(React.js) and Python. I’ve contributed to backend-heavy teams, developing high-traffic APIs and integrating with third-party services like Twilio, MailGun, SendGrid, Shopify, and Zoho.

I have deep experience building custom WordPress plugins and themes using Elementor, Divi, Thrive, and custom frameworks. I’ve extended functionality in major plugins like Yoast and developed complete solutions for WooCommerce-based e-commerce sites.

In Laravel, I’ve handled job queues, real-time broadcasting with Pusher, Redis caching, and robust service-based application design. I follow the Service pattern regularly and am also comfortable with Repository structures. I have extensive experience with advanced Laravel features including Events, Listeners, Observers, Policies, Gates, Jobs, Queues, Notifications, Broadcasting, Middleware, Schedulers, Task Batching, Laravel Horizon, Eloquent Relationships, Model Binding, Custom Macros, Morphing, Pipeline, Dependency Injection, Form Requests, Validation Rules, and Service Providers. Batch processing, API Resources, and Sanctum for token-based auth are also part of my everyday toolkit.

I’ve solved challenges like real-time iframe-based content loading, scraping with fallback services like ScrapeBee, and X-Frame security issues. I also work with Python, Node.js, and React.js for data handling, scripting, and interactive frontends. I prefer Tailwind CSS for clean frontend design.

I'm experienced in CI/CD using GitHub Actions and Docker. I regularly deploy and manage scalable infrastructure on AWS including EC2, AMI, RDS, VPC, S3, ECS, ALB, and Lambda.
